# By Breed

A React Native (Expo) app that lists cat and dog breeds with their traits. Each breed shows features like Kid Friendly, Amount of Shedding, and more, with ratings from 1 to 5.

## Setup

```bash
npm install
```

## Run

```bash
npm start
```

Then press `i` for iOS simulator, `a` for Android, or `w` for web.

## Tech

- React Native with Expo
- Data from `cat-and-dog-breeds.json`
